Web Response Debugger

概要

Web Response Debuggerは、 指定したURLのHTMLレスポンスを取得し、表示するツールです。
Cookieの影響を受けないHTMLレスポンスを確認できます。
開発者やデザイナーが、Webページの構造や内容を迅速に確認するのに役立ちます。
このサイトは学習目的で作成されております。

使い方

  1. 上部の入力欄に、確認したいWebページのURLを入力します。
  2. 「取得」ボタンをクリックします。
    取得中はローディングインジケーターが表示されます。
  3. 取得が完了すると、指定したURLのHTMLレスポンスが表示されます。
    エラーが発生した場合は、エラーメッセージが表示されます。
